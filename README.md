## 🛠️ Sobre o Projeto
Este sistema foi desenvolvido para aplicar conceitos de arquitetura de software, persistência de dados e interface do usuário. O projeto utiliza o padrão **MVC (Model-View-Controller)** para garantir a organização e escalabilidade do código.

## 🚀 Principais Funcionalidades
*   **Autenticação:** Sistema de login seguro.
*   **Gestão (CRUD):** Cadastro, consulta, edição e exclusão de registros.
*   **Interface Responsiva:** Desenvolvida com Bootstrap para adaptação em diferentes telas.
*   **Relatórios:** Painel de resumo de dados.

## 🗂️ Estrutura de Pastas
*   `/app`: Lógica do sistema (Models e Controllers).
*   `/views`: Telas e interface do usuário.
*   `/public`: Arquivos estáticos (CSS, JS, Imagens).
*   `/config`: Configurações de banco de dados e rotas.
